## 🛠️ Cara Build & Menjalankan Bot

### 1. Clone repository
```bash
git clone https://github.com/wulanlubis/Tubes_KacauMania.git
cd Tubes_KacauMania
```

### 2. Build Main Bot (Gacor)
```bash
cd src/main-bot/Gacor
dotnet build
```

### 3. Build Alternative Bots
```bash
cd src/alternative-bots/alt-bot-1/Syududu
dotnet build

cd src/alternative-bots/alt-bot-2/Kaciw
dotnet build

cd src/alternative-bots/alt-bot-3/Kacau
dotnet build
```

### 4. Jalankan Bot
```bash
dotnet run
```
Pastikan game engine sudah berjalan sebelum menjalankan bot.
